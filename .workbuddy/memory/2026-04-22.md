## 前端镜像构建修复

- 问题：`imageBuild/frontend/Dockerfile.prod` 使用 `pnpm install --frozen-lockfile`，但 package.json 已更新依赖版本（共 44 项不匹配），pnpm-lock.yaml 未同步，导致构建报 `ERR_PNPM_OUTDATED_LOCKFILE`
- 修复：将 `--frozen-lockfile` 改为 `--no-frozen-lockfile`，并加注释说明
- 根本原因：上次优化 package.json 升级了 44 个依赖版本，未同步运行 `pnpm install` 更新锁文件
- 第二个问题：`@mui/x-date-pickers@^6.20.4` 版本不存在，v6 最新只到 6.20.2，已修正为 `^6.20.2`
- 如需恢复严格锁定：在 frontend/ 目录本地执行 `pnpm install` 生成新 pnpm-lock.yaml，再改回 `--frozen-lockfile`
