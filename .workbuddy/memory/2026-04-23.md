# 2026-04-23 Domino 前端灰屏问题修复

## 问题描述
前端显示灰屏，资源 200 OK，但 JS 执行报错：`TypeError: Object prototype may only be an Object or null: import_meta_env_placeholder`

## 问题根因
`@import-meta-env/unplugin` 在构建时没有正确替换占位符：
1. `index.html` 中有 `JSON.parse('"import_meta_env_placeholder"')`
2. 浏览器执行这段无效代码，导致整个 JS 初始化失败

## 修复方案
**移除运行时注入，改用构建时注入（DefinePlugin）**

### 修改的文件
1. `frontend/vite.config.ts`:
   - 移除 `@import-meta-env/unplugin`
   - 使用 Vite 内置 `define` 选项注入环境变量

2. `frontend/index.html`:
   - 移除 `import_meta_env_placeholder` 占位符脚本

3. `frontend/entrypoint.sh`:
   - 简化为纯 nginx 启动脚本

4. `imageBuild/frontend/Dockerfile.prod`:
   - 移除 `import-meta-env` CLI 编译
   - 移除 `jq` 依赖（不再需要）
